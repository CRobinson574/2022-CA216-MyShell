User manual for myshell.c

This is a basic shell with some basic commands.
And words or strings entered without a command will be printed back out to the screen.

Commands:
clr - clears the terminal 
dir - lists all files in the current directory, including any hidden files.
environ - lists all environment variables.
cd - prints the current directory.
quit - exits the shell.


I/O redirection is the redirection of the outputs and inputs of commands into other commands or files.
